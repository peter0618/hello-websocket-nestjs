## Description

웹소켓을 이용한 채팅 프로그램 toy project 입니다.

## Installation

```bash
$ yarn install
```

## env variables
```
CHAT_SERVER_IP=127.0.0.1
```
project root 폴더에 `.env` 파일을 생성하여 위와 같이 IP를 입력합니다.

## Running the app

```bash
# development
$ yarn start:dev

# production mode
$ yarn start:prod
```
